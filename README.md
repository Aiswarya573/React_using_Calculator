#  Login Based Calculator App (React)

This project is a modern UI-based calculator application built using React. 
The application includes a login interface and redirects users to a functional calculator upon successful login.

##  Features
- User-friendly Login UI
- Gradient modern design
- Fully functional calculator
- Button-based input (like mobile calculator)
- Responsive layout
- Smooth UI/UX experience

##  Tech Stack
- React.js
- HTML5
- CSS3 (Custom Gradient UI)
- JavaScript
